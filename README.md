# 机器学习垃圾邮件检测系统 python97

## 项目概述

本项目是基于前后端分离的架构，使用Python语言开发，前端采用Vue2框架，后端使用Django框架。系统集成了Keras神经网络库和NLTK自然语言处理工具包，实现了高效的垃圾邮件识别功能，识别率可达80%以上。

## 技术栈

- 前端：Vue2
- 后端：Python Django
- 机器学习库：Keras
- 自然语言处理：NLTK

## 功能模块

- 用户登录注册
- 模型训练
- 模型预测

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python 3.x
- 相关依赖库（Django、Vue2、Keras、NLTK）
